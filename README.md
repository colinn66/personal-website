# 个人网站

这是我的个人网站项目，用于展示个人信息、作品集和博客文章。

## 项目特点

- 现代化的响应式设计
- 个人信息展示
- 作品集展示
- 博客系统

## 技术栈

- HTML5/CSS3
- JavaScript
- React.js
- Node.js

## 开发计划

- [ ] 首页设计与实现
- [ ] 个人简介页面
- [ ] 作品集展示
- [ ] 博客系统
- [ ] 响应式布局优化

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
